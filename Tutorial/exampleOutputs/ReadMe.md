# REGRO Tutorial Output Files

These files show the outputs that should be generated by following the REGRO tutorial [here](https://github.com/phmstone/REGRO/tree/main/Tutorial).

### EricalesPresenceAbsence.tsv
The presence/absence .tsv file made based on the annotations present on GenBank.

### EricalesHeatMap.png
A visual representation of EricalesPresenceAbsence.tsv.

### updatedPA.tsv
The presence/absence .tsv file made after running REGRO in discovery mode.

### EricalesHeatMap-updated.png
A visual representation of updatedPA.tsv.
