# Example Files

### angiospermMitochondrialGenes.txt
A list of angiosperm mitochondrial genes in the correct format for gene list input.    

### angiospermPlastidGenes.txt
The default angiosperm plastid gene list included in `presenceAbsence.py`.    

### angiospermPlastidReferences.TSV
A table showing GenBank ID, species name, and family for accessions that have all genes in `angiospermPlastidGenes`.    

### bashScript.sh
An bash script that runs the whole pipeline in one go using the Ericaceae test data.    

### bryophytePlastidGenes.txt
A list of bryophyte plastid genes in the correct format for gene list input.   

### ericaceaeGenbankIDs.txt
A list of GenBank IDs for the Ericales test data set used in the [tutorial](https://github.com/phmstone/PlastidGenomeContent/tree/main/Tutorial).

### gene_alias.txt
A file that can be used to assign alternative gene spellings or synonyms to a gene in the input gene list.

### greenAlgalPlastidGenes.txt
A list of green algal plastid genes in the correct format for gene list input.   

### monilophytePlastidGenes.txt
A list of monilophyte plastid genes in the correct format for gene list input. 

### referenceIDs.txt
A list of GenBank IDs for sequences used as references in the Ericales [tutorial](https://github.com/phmstone/PlastidGenomeContent/tree/main/Tutorial).
